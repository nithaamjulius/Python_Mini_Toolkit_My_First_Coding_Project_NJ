
# Project Name

Python_Mini_Toolkit_My_First_Coding_Project_NJ/ PyPlay


## Short project description
This is a python project showcasing a python mini toolkit with 3 mini tools providing the user fun.
## Features

- Main menu
- Simple quiz game
- Number guessing game
- Even or odd checker


## Python concepts incorporated
Modules, import, aliasing, functions, function calls, main guard, variables, lists, tuples, for loop, while loop, if/elif/else, casting, input, output, string methods, random, arithmetic operators, comparison operators, augmented assignment, try/except, continue, break, return, f-strings.
## How to run this project
Save both Python files in the same folder, open that folder in your terminal and run python pyplay_main_menu.py. The main menu will appear, then you type a number to choose a mini tool and the program keeps returning to the menu until you choose exit.
